# PickMe - Anket Yönetim Sistemi

## Proje Hakkında
PickMe, kullanıcıların anket oluşturmasına ve yönetmesine olanak sağlayan bir web uygulamasıdır.

## Özellikler
- Anket oluşturma ve düzenleme
- Anket yanıtlama
- Anket sonuçlarını görüntüleme

## Teknolojiler
- ASP.NET Core MVC
- C#
- HTML/CSS
- JavaScript
- SQL
